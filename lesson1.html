終極車位比喻：從基本型別到共有設施

🚗 終極停車場比喻

Primitive: 機車/滑板車直接停在車位(Stack)。
Object: 巴士太大停在VIP車庫(Heap)，車位只放「取車單/地址」。
Static: 停車場「公用打氣機」，全場共用，不屬於任何一台車。

Java 原始碼模擬

static int carCount = 0; // 公用設施
char key = 'A'; // 小型機車
float pi = 3.14f; // 滑板車
String s = "Hi"; // 貨箱單據
Person p = new Person(); // 巴士取車單
int[] row = {1, 2}; // 連號預訂
String[] tags = {"A"}; // 單據櫃
